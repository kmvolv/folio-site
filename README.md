## Overview:

Portfolio Website built with **React 18**, **TypeScript**, and **Vite**

Will possibly buy a domain in the future because boy, that is one ugly URL.

## Tech Stack

- **Framework:** React 18.2, React Router DOM v6
- **Build Tool:** Vite 5
- **Styling:** Linaria v6 (zero-runtime CSS-in-JS), plain CSS for global styles and utilities
- **Package Manager:** Yarn
